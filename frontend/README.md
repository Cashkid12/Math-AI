# Equai AI - Frontend

React frontend for the Equai AI math solver application.

## Features

- **Modern UI**: Built with React and Tailwind CSS
- **Real-time Solving**: Instant math problem solving
- **Step-by-Step Display**: Clear visualization of solution steps
- **Graph Visualization**: Interactive function graphs
- **Responsive Design**: Works on desktop and mobile
- **Dark Theme**: Beautiful dark mode interface

## Setup

1. **Install dependencies**:
```bash
cd frontend
npm install
```

2. **Run development server**:
```bash
npm run dev
```

Frontend runs on `http://localhost:5173`

## Build for Production

```bash
npm run build
```

## Technologies

- React 18
- Vite
- Tailwind CSS
- Axios
- Modern JavaScript (ES6+)

## Usage

1. Select problem type (Algebra, Calculus, or Graph)
2. Enter your mathematical expression
3. Click "Solve Problem"
4. View detailed step-by-step solution with explanations
5. See graph visualization (for graph problems)
